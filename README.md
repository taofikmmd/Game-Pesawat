## Deskripsi

Game Pesawat adalah game aksi di mana pemain mengendalikan pesawat untuk menghindari rintangan, menembak musuh, dan bertahan selama mungkin. Game ini dibuat menggunakan **Construct 2**, sehingga dapat dijalankan di browser maupun di-export ke platform lain seperti HTML5.

Fitur utama:

* Kontrol pesawat yang responsif.
* Banyak musuh dan rintangan yang menantang.
* Skor yang meningkat seiring waktu atau musuh yang dihancurkan.
* Sound effect dan musik latar.

---

## Cara Bermain

1. Jalankan file HTML atau buka game di browser.
2. Gunakan **tombol panah** (atau WASD) untuk menggerakkan pesawat.
3. Tekan **spasi** untuk menembak musuh.
4. Hindari tabrakan dengan musuh atau rintangan.
5. Kumpulkan skor sebanyak-banyaknya.

---

## Kontrol

| Aksi           | Tombol          |
| -------------- | --------------- |
| Gerak ke atas  | Panah Atas / W  |
| Gerak ke bawah | Panah Bawah / S |
| Gerak ke kiri  | Panah Kiri / A  |
| Gerak ke kanan | Panah Kanan / D |
| Menembak       | Spasi           |

---

## Requirement

* Browser modern (Chrome, Firefox, Edge, Safari).
* Construct 2 Runtime (biasanya sudah disertakan jika di-export ke HTML5).

---

## Instalasi

1. Download atau clone repository game.
2. Buka file `index.html` di browser.
3. Mainkan game.

*(Jika game di-export ke APK atau executable, cukup jalankan file tersebut.)*
